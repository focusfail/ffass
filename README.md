# focusfail's advanced sand simulator (ffass)

A "high performance" and "advanced" particle simulator.

*ffass is pronounced as the german word "Fass" (/fas/).*

## features

- [x] Sand
- [x] Stone
- [x] Resizable window
- [x] Brush size
- [ ] Shapes
  - [x] line
  - [ ] square
  - [ ] circle
  - [ ] triangle
- [ ] Transform: zoom / pan
- [ ] Water
- [ ] Wood
- [ ] Fire
- [ ] Dynamite
- [ ] Varying particle color
